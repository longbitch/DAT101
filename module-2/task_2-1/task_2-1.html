Can we create variables and constants without assigning values to them? If not, why, and which ones (variables, constants, or both)?

- A variable creates a container where a value may be declared right away or for a location where the value may be changed later on.

- Having a constant without assigning a value defies its purpose, as the purpose for a constant is to never change the value assigned.
